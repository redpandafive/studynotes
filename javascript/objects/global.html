Global Object

  • A regular JS object whose properties are the globally available to a JS program
  • Created by the JS interpreter when it starts and adds an initial set of properties
    • Global constants like undefined, Infinity, and NaN
    • Global functions like isNaN(), parseInt() and eval()
    • Constructor functions like Date(), RegExp(), String(),Object(), and Array()
    • Global objects like Math and JSON

Referencing

Depends on the host environment
  • In web browsers, the window object is the global object for all JS code contained in the browser window it represents
  • In Node, there is a global property which is the global object itself
  • Web worker threads can refer to the global object as self
  • ES2020 adds globalThis as the standard way to refer to the global object in any context
    • This has been implemented by all modern browsers and Node

Window Object

  • In web browsers, the Window object is the global object for all JS code contained in the browser window it represents
  • This has a self referential property called window that you can use to refer to it
