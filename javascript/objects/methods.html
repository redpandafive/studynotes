Object Methods

Object Methods - Call on object instance

toJSON()

  • This is not defined on Object.prototype by default
  • JSON.stringify() looks for this method on an object it is asked to serialize

toLocaleString()

  • Default just calls toString()
  • Date and Number classes define customized versions that try to format numbers, dates, and times according to local conventions
  • Can define a custom version on your object

toString()

  • Returns a string that somehow represents the object on which it is invoked
  • This is called automatically when it needs to be converted to a stringify
    • Example is using the + operator to concatenate it to a string
  • The default version is not very informative
    • [object Object]
    • Objects can implement their own version to be more helpful

toLocaleString()

  • Default just calls toString()
  • Date and Number classes define customized versions that try to format numbers, dates, and times according to local conventions
  • Can define a custom version on your object

valueOf()

  • Called when the object needs to convert the object to something other than a string, typically a number
  • Can define a custom version on your object

Object Methods - Call on Object

Object.assign()

  • Expects two or more objects as it argument
    • First argument is the target object
      • This is modified
    • Second and subsequent objects
      • These are source objects
      • These are not modified
    • For each source object, it copies the enumerable own properties of that object into the target object
      • Properties are copied when matches are found in the source object
      • This is done so properties on the last source object are the last items to be copied
      • This includes Symbols
Overwrites everything in o with defaults
Object.assign(o, defaults);
Creates a new object, copy the defaults object's properties into it, and then override those defaults with the properties in o
o = Object.assign({}, defaults, o);

Object.create()

  • Creates a new object using an existing object as the prototype of the newly created object
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/create 
              const person = {
                isHuman: false,
                printIntroduction: function() {
                  console.log(`My name is ${this.name}. Am I human? ${this.isHuman}`);
                }
              };
              
              const me = Object.create(person);
              
              me.name = 'Matthew'; // "name" is a property set on "me", but not on "person"
              me.isHuman = true; // inherited properties can be overwritten
              
              me.printIntroduction();
              // expected output: "My name is Matthew. Am I human? true"

Object.entries()

  • Returns an array of a given object's own enumerable string keyed property [key,value] pairs
  • Similar to iterating over a for..in loop, but that look enumerates properties in the prototype chain too
  • The results can be passed to the constructor of a Map object
              const object1 = {
                a: 'somestring',
                b: 42
              };
              
              for (const [key, value] of Object.entries(object1)) {
                console.log(`${key}: ${value}`);
              }
              
              // expected output:
              // "a: somestring"
              // "b: 42"
Destructuring allows you to iterate through objects 
                const obj = { foo: 'bar', baz: 42 };
                Object.entries(obj).forEach(([key, value]) => console.log(`${key}: ${value}`)); 
                
                // exprected output: "foo: bar", "baz: 42"

Object.keys()

  • Returns an array of a given objects own enumerable property names
    • Returned as an array of strings
  • Iterates the same order that a normal loop would
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/keys 
              const object1 = {
                a: 'somestring',
                b: 42,
                c: false
              };                
              console.log(Object.keys(object1));
              // expected output: Array ["a", "b", "c"]
