Objects

  • Most fundamental datatype in JS
    • Any type in JS that is not a string, number, Symbol, true, false, null, or undefined
  • A composite type
    • aggregates multiple primitives and/or other objects
  • Unordered collection of properties, which are key/value pairs
    • Dynamic, so you can add and removed as needed
  • Basically are associative arrays
  • Since keys are usually strings (can be symbols), can say an object maps strings to values
    • Objects also inherit the properties of a prototype object as well
  • Are mutable
    • Are manipulated by reference, not by value
