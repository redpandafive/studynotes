Serialization

JSON.stringify()

  • Converts an object's state to a JSON string
  • The following can be serialized
    • Objects
    • Arrays
    • Strings
    • Finite numbers
    • true
    • false
    • null
  • The following are serialized as null
    • NaN
    • Infinity
    • -Infinity
  • Dates are serialized to ISO formatted date strings

JSON.parse()

  • Restore a JSON string to JS objects
