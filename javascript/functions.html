Functions

  • JS differs from other static languages in that functions and classes are values that can be manipulated by JavaScript programs
    • They are not just part of the language syntax

Arrow Functions

  • Shorthand syntax to define a function added in ES6
  • Seperates the argument list from the function body using =>
  • Most commonly used when you want to pass an unnamed function as an argument to another function
