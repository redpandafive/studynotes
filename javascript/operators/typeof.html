TypeOf

  • A unary operator that evaluates to a string that specifies the type of the operand
  • typeof - use to detect the type of a variable (see Types) section
  • This will not differentiate between object types - use instanceof for that

Possible Return Values

  • undefined => "undefined"
  • null => "object"
  • true or false => "boolean"
  • any number or NaN => "number"
  • any BigInt => "bigint"
  • any string => "string"
  • any symbol => "symbol"
  • any function => "function"
  • any nonfunction object => "object"
