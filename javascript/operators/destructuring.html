Destructuring Assignment

  • A compound declaration and assignment syntax
  • The right side of the assignment operator is a structured value
    • Array
    • Object
    • See examples for syntax for each type
  • The left side specifies one or more variable names that mimics array and object literal syntax
    • These are usually variables to initialize as part of const, var, let statements
  • Assignment means one or more values are extracted (destructued) from the value on the right and store into the variables on the left
Simple Examples
// Same as let x=1, y=2
let [x,y] = [1,2];
// Same as x = x + 1, y = y + 1
[x,y] = [x+1,y+1];
// Swap the value of the two variables. x now contains 3 and y contains 2
[x,y] = [y,x];
Simplifies working with functions that return arrays
function a(x,y) {
return [y,x];
}
function b(y,x) {
return [x,y];
}
// y contains 2, x contains 1
let [y,x] = a(1,2);
// r contains 1, s contains 2
let [r,s] = b(y,x);
This uses a for loop to loop over all properties of an object and use destructuring to conver those pairs from 2 element arryas into individual variables
// outputs "x 1" and "y 2"
let o = { x: 1, y: 2 };
for(const [name, value] of Object.entries(o)) {
console.log(name, value);
}
Extra variables on the left are undefined, extra values on the right are ignored.
// x == 1; y == undefined
let [x,y] = [1];
// x == 1; y == 2
[x,y] = [1,2,3];
Use commas on the left to skip certain values on the right
// x == 2; y == 4
[,x,,y] = [1,2,3,4];
Use three dots to collect unused or remaining values into a single variable
// y == [2,3,4]
let [x, ...y] = [1,2,3,4];
This assigns global functions from the Math object into variables. Variable names on the left should match the property you want to copy. Items from MATH not listed on the left are ignored.
// Same as const sin=Math.sin, cos=Math.cos, tan=Math.tan
const {sin, cos, tan} = Math;
This performs like the above code, but gives the variables on the left different names
// Same as const cosine = Math.cos, tangent = Math.tan;
const { cos: cosine, tan: tangent } = Math;
