This

  • It is a keyword, not a variable or property named
    • You cannot assign a value to it
  • A function nested within a method and invoked as a function cannot use this to access the parent object
    • Its value of this will be the global object or undefined (depending on strict mode)
    • This is a common mistake to make
    • Workarounds
      • Capture the value of this into a variable - see example
      • Use an arrow function instead of a the nested function - this will inherit this
    Example of capturing an invocation context to a variable
    // An object o.
    let o = {
    m: function() {
    let self = this;
    // At this point, this points to 0

    function f() {
    // at this point, this points to undefined or global object
    // self points to o and is accessible here
    }
    }
    };
  • Regular function expressions generally do not use this
    • This is the invocation context
    • See more notes here
  • The array or object that is the subject of the property access becomes the value for the this keyword
  • This is the basis for method chaining
    • If your method does not have a return value, consider returning this
