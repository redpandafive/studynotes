Variables and Constants

  • Variables and constants must be declared before they can be used

Pre ES6

  • Uses the var keyword

ES6 and Later

  • Uses the let and const keywords
  • Block scope
    • Only defined within the block of code in which they appear
  • Can declare multiple variables in one let statement: let i, sum;
  • Good practice to set an initial value on the same line
    • Variable will contain undefined if not
    • Constants must be initialized when you declare it
  • One coding convention capitalizes all letters in a constant identifier

Scope

  • Global scope
  • Block scope
    • Only defined within the block of code in which they appear
