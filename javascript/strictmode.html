Strict Mode

Delete Operator

  • In strict mode, the delete operator will raise an error if its operand is an unqualified identifier, such as
    • variable
    • function
    • function parameter
  • An error will also raise when trying to delete a nonconfigurable property

Eval

  • In strict mode, eval will
    • Perform a local eval with a private variable environment
    • Can query and set local varaibles
    • Cannot define new variables or functions in local scope
  • Eval is treated more like a keyword
    • Cannot overwrite eval funciton with a new variable

Variable Declaration

  • If you attempt to use an undeclared variable, you will get a reference error when you run your code
    • Outside of strinct mode, it will automatically be created as a new global variable
