Expressions

  • A phrase of JavaScript that can be evaluated to produce a value
      Computes a value, but does not alter the state of the program

Primary Expressions

  • Standalone and do not include any simpler expressions
  • These include constant (lieteral) values, certain language keywords, and variable references
    • "hello"
    • 500
    • true
    • this
    • i // evaluates to the value of variable i
    • undefined // the value of the "undefined" property on the global object
