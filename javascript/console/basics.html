Console Object and Debugging Features

Console Object

  • Provides access to the browser's debugging console

Methods

console.assert()

  • Use when you want to output a console error when certain conditions are true
  • First parameter is an assertion
  • Second parameter is the message to output when the assertion is true
This message is only printed when x does not equal to 1
console.assert(x === 1, "X is not 1");

console.clear()

  • clears the console

console.log()

  • Outputs a message to the web console
  • Multiple objects to output can be specified using comma separators

console.error()

  • Outputs a message to the web console, but it formats it as an error

console.table()

  • Displays a supplied array of objects as a table in the console window
  • You can resort by column on the table by clicking column headers
  • This is a little easier format when you need to compare the values of objects

console.time()

  • Starts a timer you can use to track how long an operation takes
  • Multiple independent timers can be started - pass a string to the function as a parameter to distinguish between them
  • console.timeEnd()
    • Stops a timer
    • When you call console.timeEnd(), the browser will output the time (in milliseconds) that elapsed since the timer was started
    • Pass a string label to specify the timer to stop
console.time("MyTimer");
..... code to time here ....
console.timeEnd("MyTimer"); // this prints the elapsed time automatically

console.warn()

  • Outputs a message to the web console, but it formats it as an warning in the console

Debugger

  • You can add a breakpoint in code using the debugger keyword
const x = 5;
debugger;
x = x + 5;
