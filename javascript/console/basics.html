Console Object

  • Provides access to the browser's debugging console

Methods

  • console.clear()
    • clears the console
  • console.log()
    • Outputs a message to the web console
    • Multiple objects to output can be specified using comma separators
  • console.time()
    • Starts a timer you can use to track how long an operation takes
    • Muliple independent timers acan be started - pass a string to the function as a paramter to distinguish between them
    • When you call console.timeEnd(), the broswer will output the time (in milliseconds) that elapsed since the timer was started
  • console.timeEnd()
    • Stops a timer
    • Once stopped, the elapsed time is automatically displayed in the console
    • Pass a string label to specify the timer to stop
