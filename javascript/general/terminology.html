Terminology

Arguments
  • An expression supplied when calling a function
  • These are being passed into the function
Array
  • An orderd collection of numered values
Constructor
  • A function that initializes a newly created object
Expression
  • A phrase of JavaScript that can be evaluated to produce a value
  • Computes a value, but does not alter the state of the program
Function
  • Named and parameterized block of JS code
  • Define it once and then invoke it as many times as needed
  • In JS, these are object
Identifier
  • Simply a name to name constants, variable, properties, functions and clases
Invocation Context
  • The value of the this keyword
  • For methods, it is the object that the method is attached to
Lexical structure
  • The set of elementary rules of a programming language
  • These descript how you write programs in that language
  • Very low level
    • What variable names look like
    • Delimiter characters for comments
Literal
  • A data value that appears directly in a program, such as 12, "hello world", true, null
Method
  • A function assigned to a property of an object
Object
  • An unorderd collection of named values
Operators
  • Act on values (the operands) to produce a new value
Parameters
  • Local variables that are part of a function signature
  • These receive argument values and used to reference them from within the function
Predicate
  • A function that returns true or false
Statements
  • Do not produce a value
  • Can alter the state of the program
Variable
  • Associates an identifier to a value
