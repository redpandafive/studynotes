Map

  • Holds key-value pairs
  • Remembers the original insertion order of the keys
  • Any value can be used as either a key or value
    • This includes both objects and primitive values
  • Keys are unique in the collection, so it may only occur once
  • Only contain entries that you put it it, nothing by default
  • It is an iterable, so you can directly iterate on it
    • Can use foreach method on the Map object
    • Can use for / of loop
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Map

Properties

size
  • Returns the number of key / value pairs in the Map object

Methods

clear()
  • Removes all key-value pairs from the Map object
delete(x)
  • Removes the item indexed by key x
  • Returns true if the element was removed
  • Returns false if the element does not exist
get(x)
  • Retrieves the value identified by key x
has(x)
  • Returns true/false if the key x exists
set(x,y)
  • Sets the value of y for the key of x
  • DO NOT USE BRACKET SYNTAX
    • This will appear to work, but it is actually setting the property on the generic object
    • Other Map methods will not detect these
