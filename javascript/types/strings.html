Strings

  • Considered a primitive in JS
  • Immutable
  • There is no special type that represents a single elements of a string (such as Char in C#)
  • Use the UTF-16 encoding of the Unicode character set
    • Strings are sequences of 16 bit values
    • The length of the string is the number of 16-bit values it contains
    • An empty string has a length of zero
  • Can be treated like a read only array in that you can index into it to access individual characters

String Literals

  • Literals are enclosed by a matched pair or single quotes, double quotes, or backticks
  • ES 5 allows multiple line strings by ending each line but the last with a backslash (\)
  • ES 6 allows multiple lines when enclosed with backticks

Comparing Strings

  • Two strings are equal if and only if they consist of exactly the same sequence of 16 bit values
  • Can be compared with greater than and lesser than operators

Template Literals

  • Strings delimited with backticks
  • Can contain JavaScript expressions
  • Final string is calculated by
    • Evaluating any included expressions
    • Converting the results of those expressions to strings
    • Combining results with the literal strings within the backticks
// greeting will contain "Hello Bob."
let name = "Bob";
let greeting = `Hello ${ name }.`;

Pattern Matching

  • RegExp object is used for regular expression pattern matching
https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/RegExp

Properties

  • length = the number of 16 bit values contained in the string

Methods

Searching a String
  • endsWith("Hello")
    • Returns boolean value if the string ends with the substring
    • ES6 method
  • includes("Hello")
    • Returns boolean value if the string contains the substring
    • ES6 method
  • indexOf("zz")
    • Accepts a substring and returns the index of the first occurence in the string
    • Returns -1 if not found
  • startsWith("Hello")
    • Returns boolean value if the string begins with the substring
    • ES6 method
Finding Portions of a String
  • substring(1,4)
    • Finds a substring of the substring
    • First parameter is first character to include
    • Second parameter is the number of charaters to include
    • "Hello World".substring(1,4) returns "ell"
  • slice(1,4)
    • Similar to substring
  • split(" ,")
    • Returns an array of substrings after splitting on the specified delimiter
Padding Strings
  • padStart(5)
    • Pads the start with spaces until it reaches the specified length (first paramter)
  • padStart(5, "*")
    • Pads the start with the second parameter until it reaches the specified length (first paramter)
  • padEnd(5)
    • Pads the end with the second parameter until it reaches the specified length (first paramter)
  • padEnd(5, "*")
    • Pads the end with the second parameter until it reaches the specified length (first paramter)
Trimming Strings
  • trim()
    • Removes spaces from the start and end of the string
  • trimStart()
    • Removes spaces from the start of the string
  • trimEnd()
    • Removes spaces from the end of the string
Assorted Methods
  • at(idx)
    • Returns the character at the specifed index
    • Accepts negative numbers, which count back from the last string character
  • "test".repeat(5)
    • Concatenates 5 copies of the specified string
    • "testtesttesttesttest" is the result
See https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String#instance_methods
