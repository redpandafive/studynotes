Set

  • A collection of values like arrays, but:
    • Do not allow duplicates
    • Remembers the original insertion order of the keys
  • Values are unique in the Set, so they may only occur once. Therefore:
    • Items are either a member or not
    • It is not possible to find the number of times an item will occur in a set
  • Sets only contain entries that you put it it,so nothing by default
  • It is an iterable, so you can directly iterate on it
    • Can use foreach method on the Map object
    • Can use for / of loop

Construction

  • Can be initialized with
    • No parameters
    • An array of elements
    • Any iterable object, including another set
let s = new Set();
let t = new Set([1, s]);
let u = new Set(s); // A new set that copies the elements of s.
let unique = new Set("Mississippi"); // 4 elements: "M", "i", "s", and "p"

Iteration

Set has a forEach method to iterate over a set
  • This applies a function to each item in the set in insertion order
function logSetElements(value1, value2, set) {
console.log(`s[${value1}] = ${value2}`);
}

new Set(['foo', 'bar', undefined]).forEach(logSetElements);
Can use a for / of loop to iterate over each element in the set
let text = "Na na na na na na na na Batman!";
let wordSet = new Set(text.split(" "));
let unique = [];
for(let word of wordSet) {
unique.push(word);
}
unique // => ["Na", "na", "Batman!"]

Enforcing Uniqueness

  • One use case for Sets is when you need uniqueness in a collection of values
  • When you need unique values from an array, consider running the array through a Set to remove duplicate values

No Sparseness

  • Deleting items an Array creates spare gaps in the array since it does not automatically resize when items are removed
  • Sets do not contain spare elements; removing an element reduces the size by one

Properties

size
  • Returns the number of values in the Set object

Methods

add(x)
  • Adds the item x if it does not already exist
  • Only takes a single argument
  • Returns the set it is invoked on, so you can chain calls
clear()
  • Removes all items from the Set object
delete(x)
  • Removes the item x if it exists
  • Returns true if the item was removed
  • Returns false if the item does not exist
foreach()
  • Executes a provided function once for each value in this set, in insertion order.
has(x)
  • Returns true/false if the item x exists in the set
  • Optimized for membership testing, so this method is very fast on very large sets
values()
  • Returns a new iterator object that yields the values for each element in the Set object in insertion order

WeakSet

  • Does not prevent garbage collection from removing object that no longer have references
  • Not frequently used
  • Has the following differences with Set
    • Does not allow primitive values as members since they are not garbage collected
    • Only has add, has, and delete methods and is not iterable
    • Does not have a size property
