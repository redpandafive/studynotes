Boolean

  • Possible values are true and fakse
  • Any JavaScript value can be converted to a boolean value

    • Truthy and Falsy

    • Any JavaScript value can be converted to a boolean value
    • Falsy values - the following convert to (and function like) false
      • undefined
      • null
      • 0
      • -0
      • NaN
      • "" (the empty string)
    • All other values are Truthy values
    • This allows you to write if(0)... instead of if(0 !== null) ...
