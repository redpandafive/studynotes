Type Conversions

Implicit Type Conversion

  • JS is very flexible with type conversions and can perform them automatically
  • Some automatic conversions are unexpected
    • Strings to numbers convert unless there any non-space characters that are not part of a numeric literal. This will produce NaN
    • Boolean converts to 1 (from true) and 0 (from false)
  • You can take advantage of implicit conversions to explicitly perform a conversion
    • Append the + operator to convert a string to a number
      • Let aNumber = 5;
      • Let aString = +5;
    • Subtract zero to conver to a number
      • x - 0
    • Use the unary operator to convert the operand to Boolean
      • let aBooleanVar = !aVariable
      • let aBooleanVar = !!aVariable

Explicit Type Conversion

Explicit conversion functions
  • Boolean(), Number(), String()
  • Calling these with the new keyword will return a wrapper object leftover from early JS days - no good reason to use these
  • Number global object has two methods that provide more flexible converstions
    • parseInt()
    • parseFloat()
    • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Number
