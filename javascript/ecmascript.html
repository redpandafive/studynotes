Array methods

  • filter()
    • Creates a shallow copy of a portion of a given array,
    • Filtered down to just the elements from the given array that pass the test implemented by the provided function.
    • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter
    const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present']; const result = words.filter(word => word.length > 6); console.log(result); // expected output: Array ["exuberant", "destruction", "present"]"
  • find()
    • Returns the first element in the provided array that satisfies the provided testing function.
    • If no values satisfy the testing function, undefined is returned.
    • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find
    const array1 = [5, 12, 8, 130, 44]; const found = array1.find(element => element > 10); console.log(found); // expected output: 12 "
  • foreach()
    • Executes a provided function once for each array element
    • Doesn't actually return anything
    • Use to mutate an array or do something with each value in it
    • Slower than map()
    • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach
    const array1 = ['a', 'b', 'c']; array1.forEach(element => console.log(element)); // expected output: "a" // expected output: "b" // expected output: "c""
  • includes()
    • Determines whether an array includes a certain value among its entries, returning true or false as appropriate.
    • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/includes
    "
  • map()
    • Creates a new array with the results of calling a provided function on every elment in teh calling array
    • Supports immutability since you don't modify the existing array
    • Faster than foreach()
    • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map

Operators

  • typeof - use to detect the type of a variable (see Types) section

Spread Syntax

  • New syntax added in ES6
  • Takes an array and examps it into individual elements
  • Can be used to
    • Make shallow copies of JS objects
    • Append one array to another
    • Create a copy of an array

To spread elements of an array to function arguments

function multiply(number1, number2, number3) { console.log(number1 * number2 * number3); } let numbers = [1,2,3]; multiply(...numbers);"

To copy an array

let array2 = [...array1];"

To append an array to another without modifying the original arrays

let desserts = ['cake', 'cookie', 'donut']; let desserts1 = ['icecream', 'flan', 'frozen yoghurt', ...desserts];"

Types

  • There are five data types that can contain values
    • string
    • number
    • boolean
    • object
    • function
  • There are six types of objects
    • Object
    • Date
    • Array
    • String
    • Number
    • Boolean
  • There are two data types that cannot contain values
    • null
    • undefined
  • typeof operator to find out what data type a JavaScript variable is
    • This will return a string indicating the data type
    • const isBoolean = val => 'boolean' === typeof val;
    • Examples
      • typeof "testString" => "string"
      • typeof 9753 => "number"
      • typeof true => "boolean"
      • typeof [1,2,3,4] => "object"
      • typeof function() {} => "function"
      • typeof null => "object"
      • typeof NaN => "number"

Type Conversion

  • Append the + operator to convert a string to a number
    • Let aNumber = 5;
    • Let aString = +5;

Object.keys

  • Returns an array of a given objects own enumerable property names
    • Returned as an array of strings
  • Iterates the same order that a normal loop would
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Object/keys
