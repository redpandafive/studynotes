With

  • Runs the block of code as if the properties of a specified object were variables in scope for that code
  • Creates a temporary scope with the properties of object as variables and then executes statement within that scope
    with (object)
    statement
  • Common use case is to make it easier to work with deeply nested hierarchies
    • You can accomplish the same thing by assigning properties to variables
      with(document.forms[0]) {
      name.value = "";
      address.value = "";
      email.value = "";
      }

      let f = document.forms[0];
      f.name.value = "";
      f.address.value = "";
      f.email.value = "";
  • Forbidden in strict mode
  • Considered deprecated in non-strict mode
  • Avoid using it
    • Difficult to optimize and will run significantly slower that equivalent code written without it
