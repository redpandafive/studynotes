Control Statements

While

  • Evaluates an expression at the start of the loop and executes the statement until the expression if falsy

Do / While

  • Like a while loops, but the expression is tested at the bottom of the loop
  • Body is executed at least once

For

  • Simplifies loops that follow a pattern:
    • Initialize a counter before it starts
      • This is done only once
      • Can be an expression with side effects (assignment) or a variable declaration statement
      • Allows you to declare and initialize a counter at the same time
    • Test the counter before each iteration
    • Update the counter at the end of the loop body
      • Must also be an expression with side effects to be useful, such as an assignment or ++, --
    for(initialize ; test ; increment)
    statement
  • This is also the only place that the comma operator is commonly used in JS
    • Can combine multiple initialization and increment expressions into a single expression
    let i, j, sum = 0;
    for(i = 0, j = 10 ; i < 10 ; i++, j--) {
    sum += i * j;
    }

For / Of

  • Works with iterable objects (string, arrays, sets, and maps)
    • Cannot be used with an object, but you can iterate over Object.keys() and Object.values()
  • This loop is said to iterate over the iterable, operating on each item, one by one, in sequential order
    • Loops over an iterable object, invoking the Symbol.iterator property on the iterable
      • This returns an interator object
      • The loop will call the interactor's next() method to produce the values to be assigned to the variable
      • This will continue until
        • The iterator's next method returns an object containing done: true
        • A break exists the loop
        • a return statement runs
  • This loop returns a list of values of the object being iterated
    • Compare with for..in loops that returns a list of keys on the object being iterated 
        let list = [4, 5, 6];
                      for (let i in list) {
                        console.log(i); // "0", "1", "2",
                      }
                      for (let i of list) {
                        console.log(i); // 4, 5, 6
                      }
Syntax breakdown

  • Variable receives a value from each iterator.

    You can use a constant declaration as the variable as long as a new value is not assigned within the loop

    for(const datum of data) { console.log(property); }
  • iterable is an iterable object that is operated on
  • statement is a block statement executed for every iteration
for (variable of iterable) { statement }

For / In

  • Enumerates over all enumerable string properties of an object
    • This includes inherited enumerable properties from its prototype chain
    • Does not include symbol properties
  • Execution steps
    • Evaluate teh object expression
    • If null or undefined, skip loop
    • Evaluate the variable expression and assign the name of the property to it
    • Execute the body once for each enumerable property of the object
  • To remember its usage, think "for..in keys == foreign keys == use for..in for keys"
  • You can use a for..in loop with arrays, but it is better to use for, foreach, or for of loop instead
    • These other methods avoid non index properties
  • You can use a constant declaration as long as a new value is not assigned within the loop
    for(const property in object) { console.log(property); }
