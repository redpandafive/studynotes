Conditional Statements

  • Execute or skip other statements depending on the value of a specified expression
  • These are decision points in our code, also known as branches

If

  • Basic form contains an expression. If the expression if truthy, the statement is executed. If falsy, then the statement is not executed.
    • Parentheses around the expression are required
    • Use a statement block to combine multiple statements into one
    • An
      • else statement can be added to execute when the expression if falsy
    if (expression) statement
  • else if not a JavaScript statement, but a programming idiom that results when repeated if/else statements are used
    • Allows a multi-way branch
    • Prevents having to write deeply nested statements
    • if (n === 1) {
      // Execute code block #1
      } else if (n === 2) {
      // Execute code block #2
      } else if (n === 3) {
      // Execute code block #3
      } else {
      // If all else fails, execute block #4

Switch

  • Handles a multi way branch when all branches depend on the value of the same expression
  • To pick a branch
    • Computes the value of the expression
    • Looks for a case label whose expression evaluates to the same value (determined by ===)
      • Avoid case expressions that have side effects
    • If a match is found, execute the block of code at the statement labelled by the case
      • Break statements are jump statements that cause the interpreter to jump to the end of the switch block
      • The interpreter will continue executing statements from subsequent cases if no break statement is encountered
      • Another option is to use return statements to return from the enclosing function - no break statement needed
    • If no match is found, look for the statement labelled default and execute it
    • If no default is found, do nothing
  • One trick you can do is wrap each case within its own set of curly branches
    • This creates a local scope within the case statement, so you can redeclare constants without error
switch(n) {
case 1: // Start here if n === 1
// Execute code block #1.
break; // Stop here
case 2: // Start here if n === 2
// Execute code block #2.
break; // Stop here
case 3: // Start here if n === 3
// Execute code block #3.
break; // Stop here
default: // If all else fails...
// Execute code block #4.
break; // Stop here
}
