Editing Array Elements

Reading / Writing Array Elements

  • Access an array element to read/write values by using the [] operator
    • Array identifier should be on the left side of the brackets
    • An array index should be a non-negative integer expression
    • Index should be in the brackets
  • Arrays are just specialized objects
    • The expression used an an index is converted into a string
      • That is used as a property name
  • When using an expression that is an in-range, non-negative integer, it is treated as an array index
    • You can pass a non-negative integer as an integer or a string and it will be treated as an array index
    • The array automatically maintains the value of the length property for you
  • When using an expression that is not an in-range integer, it is treated as a regular object property
  • There is no notion of an out-of-bounds error with arrays
    • When you try to query a nonexistent property of an array, you get undefined
    • This is the same behavior as objects
    • Examples
    let i = 2; a[i] = 3;
    let a = ["world"];
    a[i + 1] = "hello";

Deleting an Array Element

  • Use the delete operator to delete the specified object property
  • This does not change the length of the array
    • This will remove the property, not just set its value to undefined
    • Does not shift elements with higher indexes down to fill the gap left by the deleted property
    • The array becomes sparse
  • Primarily used for side effect, not the value it returns
  • Only works when the operand is a property access expression
let a = [1,2,3];
// Delete the last element of the array
delete a[2];
// => false: Array element 2 doesn't exist anymore
2 in a
// => 3: Array length doesn't change
a.length
