Iterating Over Arrays

for/of

  • This will stop on gaps in sparse arrays, so make sure you test if you are working with a sparse array
  • See For / Of Loop

forEach()

  • Offers a more functional approach
  • See For / Of Loop

For loop

  • In performance critical code, you can capture the array length once in a constant so it is only referenced once on the Array object
  • You should test for undefined or nonexistant elements in sparse arrays
