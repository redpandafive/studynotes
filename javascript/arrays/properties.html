Array Properties

length

  • This property distinguishes arrays from regular JS objects
  • For non-sparse arrays, this is the number of elements in the array
    • Zero based, so the length value will be one more than the last element in the array
  • For sparse arrays, the length is greater than the number of elements
    • Can guarantee that the length value is greater than the index of every element in the array
  • You can assign a value to the length property
    • Increasing the length does not add any new elements, just creates a sparse area at the end
    • Decreasing the length removes elements that are outside of the boundaries
