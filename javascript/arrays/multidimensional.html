Multi-Dimensional Arrays

  • JS does not support true multi-dimensional arrays, but you can approximate them with arrays of arrays
  • To access a single value, use the [] operator twice
let table = new Array(10);
for(let i = 0; i < table.length; i++) {
table[i] = new Array(10);
}
// to access a single element, use
table[5][7]
