Creating Arrays

Array Initializers / Array Literals

  • Expressions whose value is a newly created array
    • These are not primary expressions
  • Arrays are initialized to the values of the supplied comma separated expressions
An array with no elements
let empty = [];
An array with 5 numeric elements
let primes = [2, 3, 5, 7, 11];
An array with 3 elements of various types and trailing comma
let misc = [ 1.1, true, "a", ];
Values can be arbitrary expressions
let base = 1024;
let table = [base, base+1, base+2, base+3];
This creates a 5 element sparse array
let sparseArray = [1,,,,5];
This creates an array with no elements, but a length of 2
let undefined = [,,];

Array Constructor

Create an empty array with no elements (similar to array literal [])
let a = new Array();
Pass a single numeric argument to specify a array length with no values when you know in advance how many elements will be required
let a = new Array();
Alternatively, the ES6 Array.of() factory function can be used. Useful when an array consisting of a single numeric argument is needed.
Array.of(10);
Pass a comma separated list of values. An array literal is simpler to use.
let a = new Array(1,2,3);

Spread Operator

  • See Spread Operator
Can be used to include the elements of one array within an array literal
// b == [0, 1, 2, 3, 4]
let a = [1, 2, 3];
let b = [0, ...a, 4];
Can also create a shallow copy of an array. The copy can be modified without changing the original.
let original = [1,2,3];
let copy = [...original];
