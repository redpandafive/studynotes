Sorting

  • Sorts elements of an array in place
  • If no parameters are passed, it sorts alphabetically into ascending order (converting elements to strings, if necessary)
  • To sort any other way, pass a comparison function as an argument
    • Comparison function takes two parameters
    • Returns negative number if first should appear before the second
    • Returns positive number if first should appear after the second
    • If they are equivalent, zero is returned
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/sort

Boosting Performance and Pitfalls

  • Avoid mutating objects while you are sorting them, this can lead to bugs or unexpecetd results
  • NaN values will always be sorted to the end of the array, so handle these in your sort function if working with arrays that contain them
    • Use isNaN(val) function to do this
  • This method may not work well with large arrays. See this article to use caching to improve performance

Examples that Sort Numbers

To sort by numerical values...
a.sort(function(a,b) {
return a-b;
});
To sort by numerical values in descending order...
a.sort(function(a,b) {
return b-a;
});

Examples that Sort Strings

To sort alphabetically, but case insensitive...
a.sort(function(s,t) {
let a = s.toLowerCase();
let b = t.toLowerCase();
if (a < b) return -1;
if (a > b) return 1;
return 0;
});

Examples that Sort Objects

Sorting by a numeric property
const users = [
{ name: 'Tim', age: 25 },
{ name: 'Frank', age: 30 },
{ name: 'Kevin', age: 20 }
];

users.sort((a, b) => a.age - b.age);
Sorting by a text property
const users = [
{ name: 'Tim', age: 25 },
{ name: 'Frank', age: 30 },
{ name: 'Kevin', age: 20 }
];

users.sort((a, b) => a.name.localeCompare(b.name));
Sorting by multiple properties
const users = [
{ name: 'Tim', age: 25 },
{ name: 'Frank', age: 30 },
{ name: 'Kevin', age: 20 }
];

users.sort((a, b) => {
if (a.age !== b.age) {
return a.age - b.age;
}
return a.name.localeCompare(b.name);
});
