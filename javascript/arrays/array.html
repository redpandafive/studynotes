Arrays

  • An ordered collection of values
    • Values in the array are called elements
    • Each element has a numeric position in the array, known as its index
    • Index numbers are 32 bit and can range from 0 to 4,294,967,294
  • Specialized form of a JS object
    • Array indexes are really little more than property names that happen to be integers
    • However, arrays are optimized, so access to indexes is generally significantly faster than access to regular object properties
  • Untyped
    • Can contain any type, including a objects or other arrays
    • Can be mixture of types
    • ES6 adds new array classes that can be typed
  • Dynamic
    • Grow and shrink as needed automatically
    • No need to declare a fixed size or reallocate it if more elements are needed
  • Can be sparse
    • Elements need not be contiguous, so you can have gaps
    • The value of the length property will be greater than the number of elements
