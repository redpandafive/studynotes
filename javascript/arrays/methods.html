Array Properties and Methods

Properties

length

  • This property makes arrays different form regular JS objects
  • For non-sparse arrays, the number of elements in the array
    • The length will be one more than number of elements in the array
  • For sparse arrays, length is greater than the number of elements
    • Can guarantee that the length value is guaranteed to be larger than the index of every element in the array
  • You can assign a value to the length property
    • Increasing the length does not add any new elements, just creates a sparse area at the end
    • Decreasing the length removes elements that are outside of the boundaries

Factory Methods

Array.from()

  • Expects an iterable or array like object as its first argument
  • Returns a new array that contains the elements of that object
  • When an iterable is passed, it works like the spread operator
    • Simple way to make a copy of an array
  • Some web browser methods use array-like objects as return values
    • Have a numeric length property and values stored with properties whose names are integers
    • Easier to work with by converting to a JS array object
  • The optional second parameter accepts a function
    • Each element from the source object is sent through the function
    • Similar to map, but more efficient that building up the array, then mapping it into another.

Array.of()

  • Useful when you need to create an array with a single numeric element. Trying something similar with the Array constructor, it will consider the number to be a length, not a value
// returns an empty array Array.of(); // returns an array with a single value, 10 Array.of(10) // => [10]; // returns an array of [1,2,3] Array.of(1,2,3)

Methods

filter()

  • Creates a shallow copy of a portion of a given array,
  • Filtered down to just the elements from the given array that pass the test implemented by the provided function.
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/filter
const words = ['spray', 'limit', 'elite', 'exuberant', 'destruction', 'present']; const result = words.filter(word => word.length > 6); console.log(result); // expected output: Array ["exuberant", "destruction", "present"]"

find()

  • Returns the first element in the provided array that satisfies the provided testing function.
  • If no values satisfy the testing function, undefined is returned.
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/find
const array1 = [5, 12, 8, 130, 44]; const found = array1.find(element => element > 10); console.log(found); // expected output: 12 "

foreach()

  • Executes a provided function once for each array element
  • Doesn't actually return anything
  • Use to mutate an array or do something with each value in it
  • Slower than map()
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/forEach
const array1 = ['a', 'b', 'c']; array1.forEach(element => console.log(element)); // expected output: "a" // expected output: "b" // expected output: "c""

includes()

  • Determines whether an array includes a certain value among its entries, returning true or false as appropriate.
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/includes
"

map()

  • Creates a new array with the results of calling a provided function on every element in the calling array
  • Supports immutability since you don't modify the existing array
  • Faster than foreach()
  • https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/map
const numbers = [1, 4, 9];
const roots = numbers.map((num) => Math.sqrt(num));

// roots is now [1, 2, 3], numbers is still [1, 4, 9]

Adding / Removing Elements

  • pop()
    • Removes the last element of the array and returns it
    • This reduces the length of an array by 1
  • push()
    • Adds an element to the end of the array
    • Adds the new element to array[array.length]
  • shift()
    • Removes and returns the first element of the array
    • This reduces the length of an array by 1
    • Shifts all elements down to an index one lower than the current index
  • splice()
    • General method for inserting, deleting, or replacing array elements
  • unshift()
    • Adds an element to the beginning of the array
