Configuring Git from CLI

  • Commands
  • To configure your instance of Git, use the git config command
    • This information is sent with your commits
    • The global flag sets this for all repositories on the workstation
    • You can overwrite these settings for a single project by running the same commands again from the project folder without the global option. This only impacts that project.
  • git config --global user.name "Colin Speight"
    • Configures the usename
  • git config --global user.email "cspeight@gmail.com"
    • Configures your email
  • git config
    • Displays all configured information
  • git config --global core.editor "'C:/Program Files/Notepad++/notepad++.exe'"
    • Use this to configure Windows Git Bash to use Notepad++ as the editor
