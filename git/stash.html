Stash

  • Commands
  • Stash allows you to save the state of your working directory and staging
  • This is not a commit, so it is not recorded as one
  • After completing, the working directory and staging are restored to align with the last commit
  • You can restore from the stash at a later point to resume
  • This will stash files that are in the working directory but are not tracked by Git

Commands

  • git stash = puts the working directory and staging state into stash
  • git stash list = lists WIP progress changes that have stashed, including the branch and commit informaiton
  • git stash show = shows details of changes that are recorded in the stash entry
  • git stash pop
    • Restores changes that are stashed
    • Staged changes appear to revert back to working directory when this is done
