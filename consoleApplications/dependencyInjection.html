Dependency Injection

.NET

  • IOC containers can be wired up as a DependencyResolver

.NET Core

  • ASP.NET Core includes its own IOC container
  • Support for dependency injection in console applications can be added via NuGet
    • Microsoft.Extensions.DependencyInjection
  • Add the code to the Main method to provide the mapping to the IOC container
