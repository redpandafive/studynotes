In a Conditional

  • Extends can be used in a conditional to check if either of the following is true:
    • Proper subset = Every element of the left side "set" is also in the right side "set"
    • Subset = Every element of the left side "set" is in the right side "set" and the right side has no extra elements

Generic Constraint

  • Constraints limit the kinds of types that a type parameter can accept
The following constraint requires the type to have a length property
  • Without this, you would not be allowed to access the length property in the function
function longest<Type extends { length: number }>(a: Type, b: Type) {
if (a.length >= b.length) {
return a;
} else {
return b;
}
}

Interface Declaration

  • The object type can be seen as the set of all possible object shapes
  • An interface can be seen as the set of all possible object shapes whose properties match the parent interface
    • This makes an interface a subset of the object type
  • When extending an interface, the child interface is a proper subset of the parent interface

Examples

Since both 'a' and 'b' are strings, they are subsets of the type string
// true
type W = 'a' extends string ? true : false;
// also true
type X = 'a' | 'b' extends string ? true : false;
Example use as a generic constraint. Only types T that are subsets of the string type are permitted.
