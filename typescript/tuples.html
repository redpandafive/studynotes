Tuples

  • A typed array with a pre-defined length and types for each index
  • All items in the Tuple do not need to have the same type
    let tup: [number, boolean, string];
    tup = [5, false, 'some string value'];
  • Tuples can be made readonly for safety
    const tup: readonly [number, boolean, string] = [5, false, 'some string value'];
  • Tuples can be named to describe values at each index
    const graph: [x: number, y: number] = [55.2, 41.3];
  • Tuples can be destructured since they are arrays
    const graph: [number, number] = [55.2, 41.3];
    const [x, y] = graph;
