Set Theory

  • A branch of mathematics devotes to collections of elements, or sets
  • Sets can be described by:
    • Enumerating the elements it contains:
      • {a,b,c}
    • Stating a rule to determine which elements belong in the set
      • {x ∈ S | P(x)}
      • "The set of all elements x belonging in set S such that P of x"
      • P function is called predicate and selects elements that belong to the set
        • It is the constraint that every element must fulfill in order to qualify as a member in the set
  • Once a set is defined, elements can be described as members or not members of the set
    • a ∈ {a, b, c}
    • d ∉ {a, b, c}
  • A set is a subset of another set if, and only if, every element of the first set is also an element of the second set
    • When all items in one set exist in the other set, then the first is a subset of the second
    • {a, b} ⊆ {b, a}
  • Given two sets, if there is at least one element in the second set that does not exist in the first set, then
    • The first (smaller) set is a proper subset of the second (larger) set
    • The second set (larger) is a superset of the first (smaller) set
    • {a, b} ⊂ {a, b, c}
  • Two sets are considered equal if, and only if, they contain the same elements:
    • Every member of the first set is also a member of the second set
    • Every member of the second set is also a member of the first set
  • A set is not a subset of another set if there exists at least one element in the first set that is not a member of the second set
    • Overlapping sets contain at least one element in both sets
    • Disjoint sets contain no elements that are in both the first and second sets
  • The empty set is a special case set
    • Symbolized as ∅
    • Contains nothing. It itself is not nothing, rather it is a container of nothing
    • Given two sets that are disjoint, the elements common to both is the empty set
  • The universal set is the set that every set is a subset of, and that every element is a member of
    • Symbolized by ∪
