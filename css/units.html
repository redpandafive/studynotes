Units of Measurement

  • Font size is a cascading property, meaning it will just flow down the chain of elements until it is re-defined

Relative Units

  • Em
    • These tend to not work as well in Angular components since you may not be able to predict the size of the host where the component will be used
  • Rem
    • In Angular, this measurement presents a good opportunity to have the component be consistently rendered the same size, no matter the size of the host
    Add this to the stylesheet of each component
    :host {
    font-size: 1rem;
    }
