Basics

Rules

  • display
    • Note, this rule cannot be set on an element with display: inline
  • height
    • Note, this rule cannot be set on an element with display: inline
  • box-shadow
    • Adds shadow effects around an elements frame
    • Described by
      • X and Y offsets relative to the element
      • Blur
      • Spread radius
      • Color
      • https://developer.mozilla.org/en-US/docs/Web/CSS/box-shadow
  • width
    • Note, this rule cannot be set on an element with display: inline
  • white-space
    • Sets how white space inside an element is handled
    • Specifies if and how white space is collapsed and whether lines may wrap at soft wrap opportunities
    • https://developer.mozilla.org/en-US/docs/Web/CSS/white-space

Pseudo Selectors

  • :empty
    • Select an element that does not contain any children, including:
      • Element nodes
      • Text
      • Whitespace
    • This does not consider comments to be content
  • :Host
    • Useful in Angular to reference a component template element from within itself
