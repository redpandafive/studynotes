Selectors

Basic

  • Universal selector
    • *
    • Selectes all elements
  • Type selector = such as input
  • Class selector
    • Select elements with the given class attribute
    • .index matches any element that has class="index"
  • ID selectors
    • Finds the element with the given id
    • IDs should be unique among elements
    • #idname will match id="idname"

Attribute

  • Finds elements that have the given attribute
  • Can optionally add a value expresion to filter by value
  • Matches are case sensitive
  • The following find all elements that have the attr attribute that has a value that:
    • [attr] - contains any value or no value
    • [attr="value"] - exactly matches "value"
    • [attr~="value"] - is contained in the specified whitespace serated list of words
    • [attr|="value"] - exactly matches "value" OR begins with "value" immediately followed by a hyphen
    • [attr^="value"] - begins with "value"
    • [attr$="value"] - ends with "value"
    • [attr*="value"] - contains "value" anywhere in it (substring)
  • https://developer.mozilla.org/en-US/docs/Web/CSS/Attribute_selectors

Grouping

  • Selector list
    • Selector is a grouping method that selects all the matching nodes
    • div, span finds all span and div elements

Combinators

  • Descendant combinator
    • Select all nodes that are descendants of the first element
    • Represented as a space
    • div span will match all span elements that are inside of a div element
  • Child combinator
    • Selects nodes that are direct children of the first element
    • Represented by >
    • ul > li will match all li elements that are nested directly inside a ul element
  • General Sibling combinator
    • Second element follows the first and both share the same parent
    • Do not have to be adjacent to match
    • Represented by ~
    • p ~ span will match all span elements that follow a p, immediately or not
  • Adjacent Sibling combinator
    • Matches the second element only if it immediately follows the first element
    • Must be adjacent and have the same parent to match
    • Represented by +
    • p + span will match the first span element taht immeidately follows a p element
  • Column combinator
    • Selects nodes that belong to a column
    • col || td will match all td elements that belong to the scope of the col
