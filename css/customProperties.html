CSS Variables (Custom Properties)

  • Defined by CSS authors that contain specific values to be reused throughout a document
    • Allow you to define a value to be stored in one place, then reference it multiple other places
    • Also allows semantic identifiers so you can identify a value easier
    • These are not actual variables like other languages, so values are computed where it is needed, not stored for use in other rules
  • These are subject to cascade and inherit their value from their parent
  • Defined using the custom property notation
    • Prefix with --
    • Define within a ruleset. Common best practice is to define them on the :root pseudo-class so it can be applied globally across the HTML document
    :root {
    --main-bg-color: brown;
    }
  • Access the value using the var() function
    • This can also take a second value (or more) to use as a fallback if the specified custom property cannot be found
    .five {
    background-color: var(--main-bg-color);
    }
    Example that includes fallbacks
    .three {
    background-color: var(--my-var, var(--my-background, pink));
    }
