Stacking Context

  • 3D conceptualizatoin of HTML elements along an imaginary z-index relative to the user's perspective
  • Overlapping elements are stacked in priority order based on element attributes
  • Use the z-index property on a positioned element to create a custom stacking order
  • Layer description (X is an arbitraty integer)
    • Bottom Layer
      • Layer farthest from the observer
    • Layer -X
      • Layers with negative z-index values
      • Layer 0
      • Default rendering layer
      • Layer X
      • Layers with positive z-index values
      • Top Layer
      • Layer closets to the user
