Naming Conventions

  • There have been several CSS naming conventions developed over the years to give a consistent naming convention for CSS
  • One approach is B.E.M
    • This stands for class="block__element--modifier"
  • Block is just a related chunk of HTML elements
  • Element is a single element within a block
  • Modifier modifies the element
Simple example
<ul class="nav"> <li class="nav__item"> Home </li> <li class="nav__item nav__item--active"> About </li> </ul>
