Pseudo Elements

  • A keyword added to a selector that lets you style a specific part of the selected element(s)
  • Can only use one pseudo element in a sector
  • https://developer.mozilla.org/en-US/docs/Web/CSS/Pseudo-elements
  • :: after and ::before
    • Creates a pseudo element that is the first child or the last child of the selected element
    • Often used to add cosmetic content to an element with the content property
    • https://developer.mozilla.org/en-US/docs/Web/CSS/::before
    Outputs <<Some quotes>>, he said, <<are better than none.>>
    q::before {
    content: "<<";
    }

    q::after {
    content: ">>";
    }
    }
    ...
    <q>Some quotes</q>, he said, <q>are better than none.</q>
  • ::marker
    • Selects the marker box of a list item, which typically contains a bullet or number
    • Works on any element set to display: list-item, such as <li> and <summary>
    • https://developer.mozilla.org/en-US/docs/Web/CSS/::marker
    The following sets the list marker to a cross of font size 1.2
    li::marker {
    content: '‚úù ';
    font-size: 1.2em;
    }
  • ::placeholder
    • Targets placeholder text in an <input> and <textarea>
    • https://developer.mozilla.org/en-US/docs/Web/CSS/::placeholder
    The following styles placeholder text of input elements to red, bold, and .5 opacity
    input::placeholder {
    font-weight: bold;
    opacity: 0.5;
    color: red;
    }
