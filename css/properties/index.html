Layout

  • position
    • Sets how an element is positioned in the document
    • Final position is determined by combination of top, right, bottom, and left properties
    • Terminology
      • Positioned element = element whose computed position value is anything other than static
      • Absolutely positioned element = element whose computed position value is absolute or fixed
        • Usually leave hight and wdith on auto and using top, right, bottom, and left
    • Possible values
      • absolute
        • Element is removed from the normal document flow
        • No space is creted for the elment in the page layout
        • If it has a positioned ancestor, it is positioned relative to that element
        • If it does not have a positioned ancestor, it is placed relative to the initial containing block
        • The top, riht, bottom, and left properties specify offsets from the edges of the element's containing block
        • Creates a new stacking context
      • fixed
        • Element is removed from the normal document flow
        • No space is creted for the element in the page layout
        • Positioned relative to the initial containing block established by the viewport
        • The top, riht, bottom, and left properties specify offsets from the edges of the element's containing block
        • Always creates a new stacking context
      • relative
        • Element is position ed according to the normal flow of the document
        • Properties top, right, botton, and left offset the element relative to itself
          • The top and bottom properties specify the vertical offset from its normal position
          • The left and right properties specify the horizontal offset
        • Offsets do no affect the position of any other elements
        • Creates a new stacking context
      • static
        • This is the default value
        • Element is positioned according to normal flow of the document
        • Properties top, right, left, and z-index have no effect
      • sticky
        • Positioned according to the normal flow of the document until its containing block crosses a specified threshhold within its flow root
        • Offset relative to it nearest scrolling ancestor and containing block
        • Offsets do not affect the position of any other elements
        • Creates a new stacking context
    • https://developer.mozilla.org/en-US/docs/Web/CSS/position
