Display Property

  • Sets whether an element is treated as a block or inline element and the layout used for its children
    • Specificaly, it sets an element's inner and outer display types
  • CSS Level 3 supports setting inner and outer display types seperately, but it is not well supported right now
    • I excluded all notes related to this: see https://developer.mozilla.org/en-US/docs/Web/CSS/display
  • Outer display type
    • Sets an element's participation in flow layout
    • Possible valueChanges
      • block
        • The element generates a block element box
        • Generates line breaks both before and after the element when in the normal flow
      • inline
        • The element generates one or more inline element boxes that do not generate line breaks before or after themselves
        • In normal flow, the next element will be on the same line if there is space
  • Inner display type
    • Sets the layout of children
    • Possible values
      • table - behaves like a table element
      • flex - behaves like a block element and uses flexbox model
      • grid - behaves like a block element and uses grid model
  • Box display type
    • Possible values
      • contents
        • Element doesnt produce a specific box by themselves
        • Element is replaced by their pseudo-box
      • none
        • Turns off display of element and all its children
        • Element has not effect on layout (unlike visibility)
