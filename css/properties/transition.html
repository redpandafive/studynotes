Transition

  • Instead of having changes show immediately, you can define transitions so changes are animated over time intervals and folow an acceleration curve
  • Called an implicit transition
    • It is a transition between two states
    • The Web author defines the first and last states
    • The browser calculates the in-between states
  • These are also useful to make JavaScript functionality smoother
  • Properties can be set to deterime how the transition is performed
    • When the animation will start
    • How long the transition will last
    • How the transition will run (linearly, quick at beginning and slow at start, etc)
  • Not all properties can be animated: see https://developer.mozilla.org/en-US/docs/Web/CSS/CSS_animated_properties
  • Properties
    • transition
      • A shorthand property for all transition properties
      Here are the order constituent properties are defined
      div {
      transition: <property> <duration> <timing-function> <delay>
      }
    • transition-property
      • The name or names of the CSS properties to which transitions should be applied
      • Changes not listed here are instantaneous
      • transition-property: margin-right, color; specifies only the two specified properties
      • transition-property: all; specifies all properties to transition
      • transition-property: none; specifies no properties will transition
    • transition-duration
      • The duration over which transitions should occur
      • Can specify a single duration that applies to all properties during the transition or multiple so properties can have different durations
      • Defined using a zero or positive integer followed by s or ms
        • Zero values begin immediately
      • Multiple durations can be added; each will be applied to the corresponding property as defined in the transition-property property
    • transition-timing-function
      • Specifies a function to define how intermediate values for properties are computed
      • Cheat sheet for easing function definitions
    • transition-delay
      • How long to wait between the time a property is changes and the transition actually begins
      • Defined using a zero, positive, or negative integer followed by s or ms
        • Zero values begin immediately
        • Negative values begin immediately, but partway through the effect (as if it had already been running for the given length of time)
      • Muliple dealys can be specified to transiition multiple properties at different rates
      • Examples
        • transition-delay: 3s; waits 3 seconds to begin
        • transition-delay: 2s, 4ms; delays 2 seconds for the first property to transition and 4 milliseconds for the second property to transition
