Basics

  • These are three possible approaches to get started with EF
    • Database First
      • Use when you have an existing database to map to
      • Generates a EDMX (Entity Data model XML) file that will contain all the data models
    • Model First
      • Create a EDMX file with models in the designer, then generate a database from the EDMX file
    • Code First
      • Create data models first, then generate or update the database with incremental migrations
