Model Configuration

Data Annotations

  • Data annotations can be added to the model to configure the resulting database
    The following sets the Username property to be the primary key of the User table
    public class User
    {
    [Key]
    public string Username { get; set; }
    }

Data Annotations - Attribute Tags

  • [Key] = identifies this property of the table
  • [DatabaseGenerated(DatabaseGeneratedOption.Identity)]
  • [MaxLength(100)] = sets the maximul length to 100 characters
