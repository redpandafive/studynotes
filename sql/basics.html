JSON in SQL Server

  • To format query results as JSON, add FOR JSON to the SELECT statement
  • To maintain control over teh JSON output, use FOR JSON PATH
  • To format JSON output automatically based on the structure of the SELECT statement, use FOR JSON AUTO
  • https://learn.microsoft.com/en-us/sql/relational-databases/json/format-query-results-as-json-with-for-json-sql-server?view=sql-server-ver16
  • Example: SELECT .... FROM TableName FOR JSON PATH, ROOT('order');
