Switch Expressions

  • Allows you to write switches as an expression that return a value
  • Important to remember that these should return a value, so either
    • Have a variable to the left of an assignment operator
    • Define a function that returns a value and uses => syntax with the switch
  • Syntax
    • Put the variable name to switch on, then the switch keyword
    • Do not need
      • Case keywords
      • Break keywords - no fall through
    • Use
      • Underscore character to indicate a discard value or default - this will match anything, including nulls
      • Use a lambda to return a value
      • Null
    • Can optionally use
      • Type keyword on the left to switch on the type
      • Property pattern match to match on a property value
      • ICollection<char> { Count: >= 5 } => ..."
  • Cases take other switch expressions as well as property matching statements
  • Can combine with Tuple patterns
  • The above is a snipped from a method that takes two parameters, C1 and C2
  • switch on a Tuple containing C1 and C2
  • Can add a conditional

Top Level Statements (Console Application)

  • Eliminates the need for the boilerplate code on a console application Main file
    • No static Main method
    • No program class
  • Added in C# 9
  • The eliminate code is put back in the generated IL
  • There can be only one Main method in .NET programs
  • This code that is added in Program.cs without the explicit class, method, etc. code are called Top level statements
    • Top level statements can only be used in the Program.cs file
  • Namespaces and type declarations must be after all top-level statements
    • These are compiled to IL as if they were separate files
  • The .NET 6 SDK adds a set of implicit global using directives
    • These include the most common namespaces for the project type
    • Means you do not have to add a using statement to access Console object
  • You can still access the args parameter in the top level without doing anything special
    • Just access it like it has already been declared
    • This is a string parameter
