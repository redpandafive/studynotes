JSON

  • Commands
  • C# 8 now has built in JSON support
    • JSON.NET was the standard way to serialize and deserialize JSON data
    • Since this was an external library, any applications you write and dependent on it-may lead to versioning problems wince you are tied to it
  • System.Text.Json namespace
    • Contains classes for JSON finality
    • Fast
      • JSON.NET had to work with UTF16 strings
      • New namespace works with ray Utf8 format
    • Minimizes memory allocation using the new Span datatype
    • Not as feature rich as JSON.NET
    • Don’t have to switch to this, but if your needs are simple, they are definitely faster
  • Reading JSON data with Utf8JsonReader class
    • Very low level - can be difficult to work with if you don’t this granular level control
      • Most applications will not use this
    • Read file in as bytes
    • Convert to Span
    • Loop through a line at a time
  • Reading JSON data with JsonDocument
    • More friendly way to interact with JSON data
    • Gives you a document object model object to reference items
  • Writing JSON data using Utf8JsonWriter
  • Using JsonSerializer
