Threading

  • Difficult to get right
  • Hard to unit test properly
  • Locking
    • Slightly expensive
  • Deadlock
    • Two threads each hold a locked resource that the other is trying to get
  • Mutex
    • Use to create lock

Pausing Execution

  • Thread.Sleep
    • Classic way to suspend execution
    • Suspends the current thread until the given amount of time has elapsed
    • Nothing you can do to abort this except waiting until the time elapses or restarting the application
  • Task.Delay
    • This is an async version of Thread.Sleep
    • Creates a Task that will complete after a time delay
    • This can be cancelled
    • Use the await keyword before the Task.Delay call
