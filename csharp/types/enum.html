Enums

  • Easy way to encapsulate a set of public, read only properties into a single object without having to explicitly define a type first

Cookbook

To convert an enum to the string representation of the entry, use the built in ToString() method. The following will output "Red"
enum Colors {Red = 1, Blue = 2}; Enum myColors = Colors.Red; Console.WriteLine("{0}", myColors.ToString());
