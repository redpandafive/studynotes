Interval

  • A creation operator that creates an Observable that emits a sequential number at a defined interval
  • Parameter is the number of milliseconds between emissions
  • This will continue to emit values until the Observable is stopped
  • Use cases
    • This can be used to create an asynchronous set of numbers for testing
  • There will be an initial delay of the first emission (of one interval period of time)
    • If this is not desired, consider using the timer operator as you can specify a different initial delay
Example that:
  • Delays first emission by 10 seconds (10000 milliseconds)
  • Emits 0
  • Waits 10 seconds
  • Emits 1
  • It will continue to do this until the Observable is stopped
interval(10000).subscribe(console.log);
