Of, From, FromEvent Operators

Of and From

  • These operators
    • Create an observable object around the values passed in
    • Emit each object one at a time
    • Once all items have been emitted, the complete method is called automatically
    • DO NOT NEED TO UNSUBSCRIBE FROM THESE - the complete method handles this

Of

  • Returns an observable that emits a single value
  • If you pass in an single array with 5 items, you will get 1 emission (the entire array)
    • You would have to use the spread operator (...) to get an equivalent output as the from operator
  • You can pass in a comma separated list and you get one emission for each item in the comma separated list
Emits 1 value , [2, 4, 6, 8]. The parameter to the subscribe method is the next() function.
of([2, 4, 6, 8]).subscribe(item => console.log(item))
Emits 4 values separately, 2, 4, 6, 8
of(...[2, 4, 6, 8]).subscribe(item => console.log(item))

From

  • returns an observable that will emit each item in the provided collection
  • If you pass in an array with 5 items, you will get 5 emissions (one for each item in the array)
Emits 4 values separately, 2, 4, 6, 8
from([2, 4, 6, 8]).subscribe(item => console.log(item))

FromEvent

  • Creates an Observable from any DOM event
  • It is easier to work with event binding in Angular to do this
This logs an event whenever a click event fires on the ViewChild element
@ViewChild('para') par: ElementRef;
const parStream = fromEvent(this.par.nativeElement, 'click').subscribe(console.log);

Interval

  • A creation operator that creates an Observable that emits a sequential number at a defined interval
  • This can be used to create an asynchronous set of numbers for testing
  • Parameter is the number of milliseconds between emissions
  • This will continue to emit values until the Observable is stopped
