WithLatestFrom

  • A combination operator
    • Takes in a set of Observables and subscribes to each
    • This creates an output Observable
  • When a item is emitted from the source Observable
    • If all Observables have emitted at least once
    • Emits a value to the output Observable
    • The emitted value combines the latest emitted value from each input Observable into an array
  • Values are only emiited when the outer observable emits
    • Emitted values are a collection of data that includes the values from the outer observable and the sub-observables
    • Emitted values include the latest values from all observables
    • Outer observable does not emit until all observables have a value to emit
  • If the outer observable only emits once and the inner observables only emit after the outer observable, then you will never have an emission from the overall observable
  • This completes when the source observable completes
  • This is a pipeable operator
  • Use cases
    • Use to react to changes in only one Observable
    • To regular the output of the other Observables
    • Not good for HTTP calls
