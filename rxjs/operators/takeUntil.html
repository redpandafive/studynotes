TakeUntil

  • One common pattern is to use this to automatically unsubscribe from an observable when a component is destroyed
Add a class property for Subject
private destroy$: Subject<boolean> = new Subject<boolean>();
Setup the observable in the component
this.searchFieldControl.valueChanges.pipe( takeUntil(this.destroy$)). subscribe(....do more stuff here...) )
In the ngOnDestroy method
this.destroy$.next(true); this.destroy$.unsubscribe();
