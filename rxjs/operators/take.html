Take

  • Emits the specified number of values before automatically calling complete on the observable, regardless if the source completes
  • Useful to limit unlimited observables
  • When an item is emitted, the operators
    • Counts the item
    • If <= specified number, it emits the item to the output Observable
    • When it equals the specified number, it completes
//emit value every 1s
const interval$ = interval(1000);
const example = interval$.pipe(
take(5),
map(item => mitem * 2)
);
//output: 0,2,4,6,8
example.subscribe(val => console.log(val));
