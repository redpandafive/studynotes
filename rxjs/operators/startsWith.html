StartsWith

  • A combination operator
  • Creates an output observable
    • When subscribed to, synchronously emits all provided values
    • When an item is emitted to its input Observable, the item is emitted to its output Observable
  • Starts the stream by emitting the specified value as an initial value
    • Any values emitted by the source essentially "pass through" and are emitted
  • Initial values must be the same type as the input Observable emissions
  • Useful with the valueChanges stream from Reactive Forms
    • This is a good way to provide a default value before the user actually starts typing in the control
startsWith('red')
The following will emit a, b, c, d
of('b', 'c', 'd')
.pipe(
startWith('a')
).subscribe(x => console.log(x));
