CombineLatest

  • This is a static combination, creation function
    • Not sent through the pipe method
    • This creates a new stream from scratch
  • Processes latest emitted item from each observable anytime an observable emits a new value
  • Takes in multiple streams contained in an array
    • Will always emit the latest values from all of the streams provided
    • Will not emit a value until all provided streams emit at least one value
    • Once all streams have a value, an update on any one of them will create a new
  • This completes when all included observables complete
  • Use cases
    • View models
    • To work with multiple data sets
    • To re-evaluate state when an action occurs
  • Use the descrtucturing syntax in the final map to split them apart into a format that is more convenient to work with
    combineLatest([...]).map( ([checklist, items, forMalIsOpen, checkIfItemBeingEdited]) => ({....}));
