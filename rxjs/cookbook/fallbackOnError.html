Use a Fallback Observable on Error

The following code:
  • Creates two observables from static values
  • Uses the throwError operator to creae an observable that immediately emits an error notification
  • Pipes the value emitted by observableWithError into the catchError operator
    • This returns the fallbackObservable observable
  • Since catchError has intercepted the error, the error method on subscribe is not called
import { throwError, of } from 'rxjs';
import { catchError } from 'rxjs/operators';

const observableWithError = throwError('Oops! Something went wrong!');
const fallbackObservable = of('Fallback value');

observableWithError.pipe(catchError((error) => fallbackObservable)).subscribe(
(value) => console.log(`Received value: ${value}`),
(error) => console.log(`Received error: ${error}`)
);
