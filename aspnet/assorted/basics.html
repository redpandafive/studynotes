Basics

  • Redesign of ASP.NET 4.x
  • Leaner, more modular framework
  • Unifies Web UI and Web APIs
  • Features
    • Runs on Windows, MacOS, and Linux
    • Open sourced
    • Built in dependency injection
    • Applications are standalone programs and not dependent on IIS
  • Drops support for
    • WCF
Files with leading underscore are not intended to be browsable - these could be
  • Layout pages - _Layout.cshtml
  • _ViewImports.cshtml
  • _ViewStart.cshtml
  • Partial pages
  • Alternative to ASP.NET Core MVC
    • Preferred way to create page or form-based apps in ASP.NET Core
    • Seems very reminiscent of Web Forms
    • Separates logic of a page form from its presentation
  • Shares many of the same features as MVC
    • Layout pages
    • Partial pages
    • Templates
    • Tag helpers

Configuration

To enable, this must be enabled in the Program.cs file
  • Enable support for Razor pages
    • App.AddRazorPages()
  • Enable endpoints for Razor Pages
    • App.MapRazorPages()
