Partial Pages

  • Contain snippets of HTML and server side code to be included in any number of pages or layouts
  • Use to break up complex pages into smaller units
  • Automatically inherits the ViewData associated with the calling view, so it may behave differently on differnt Razor pages
  • View Components give a more powerful alternative to these in .NET 6
