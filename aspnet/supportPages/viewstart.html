ViewStart Pages

These pages are named _ViewStart.cshtml
  • Code that is executed at the start of each Razor page execution
    • Common use case is to set the layout page for each Razor page
  • Affects all Razor pages in the same folder or any of its subfolders
  • These are hierarchical, so you can have them at different levels in the folder strucutre
  • Those located in subfolders will be executed after those located higher up the file system hierarchy
