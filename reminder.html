Reminders

JavaScript

  • Use Nullish Coalescing Operator instead of || to check for null values
  • Freeze object to make them immutable
  • Consider Sets and Maps where appropriate
  • Use Sets instead of Arrays when you need uniqueness and do not want spareness
  • Use specialized Console methods instead of always console.log
    • assert
    • warn, error
    • table
    • time
