Disable the Submit Button on an Invalid Form - Reactive Forms

  • Setup a FormGroup property in the component and setup validators
    reactiveForm: FormGroup;
    ....
    ngOnInit() {
    this.reactiveForm = this.builder.group({
    age: [null, Validators.required]
    });
  • Associate the FormGroup to the form element in the template
    <form [formGroup]="reactiveForm"></form>
  • Bind the disabled attribute on the button to the state of the FormGroup (uses same states as template version)
    <button type="submit" [disabled]="reactiveForm.invalid">Submit</button>
  • Both forms and controls have additional state properties you can reference if needed
    • myVar.valid
    • myVar.invalid
    • myVar.status
    • myVar.disabled

Disable the Submit Button on an Invalid Form - Template Forms

  • Tag the form controls with native HTML validtion attributes
    • Can also define custom validators if needed
    • https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/Constraint_validation
    • https://angular.io/guide/form-validation#custom-validators
  • Create a local template variable to reference
    • The following captures a reference to a single control
      <input type="text" id="name" name="name" required minlength="4" [(ngModel)]="hero.name" #myVar="ngModel">
    • The following captures a reference to a form
      <form (ngSubmit)="onSubmit()" #myVar="ngForm"></form>
  • Bind to the disabled attribute on the button
    • <button [disabled]="myVar.invalid">Submit</button>
    • !myVar.valid will have the same result

Resetting a Reactive Form

  • <button (click)="myForm.reset()">Reset</button>

Submitting a Reactive Form

  • Can bind to the Submit event on the form element. This passes the JSON object of input values to the addProduct method.
    <form [formGroup]="myForm" (ngSubmit)="addProduct(myForm.value)"></form>
  • Can also bind to the click event of the submit button. This passes the JSON object of input values to the addProduct method.
    <form [formGroup]=”myForm”>
    ...
    <button type="submit" [disabled]="myForm.invalid" (click)="addProduct(myForm.value)">Submit</button>
    </form>
