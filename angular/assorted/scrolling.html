Scrolling

  • Uses the ViewportScroller service as a scroll position manager

Methods

  • scrollToAnchor()
    • Use to scroll to an anchor element
    • Parameter is the ID of the anchor element to scroll to

Additional Settings

  • Set the scrolling method on the root HTML, Body tag to use native browser smooth scrolling
  • Make sure you set this in the root stylesheet, not one of the component stylesheets
html, body { height: 100%; scroll-behavior: smooth; }
