Bootstrapping

  • Loads main.ts file
    • This is referenced by the main property of angular.json to identify the startup
    • Enables Prod mode if that is enabled
    • Bootstraps AppModule with a call to platformBrowserDynamic().boostrapModule(AppModule)
  • AppModule bootstraps AppComponent
    • This is defined as a property on the NgModule decorator under the boostrap configuration property
    • Foundation component for an Angular project, so all applications require one and only one
  • App component
    • When you bootstrap with the AppComponent class (in main.ts), Angular looks for a app-root in the index.html, instantiates an instance of AppComponent, and renders it inside the app-root tag
    • The app-root tag will be in the Index.html file included in the root of the Src folder
  • Index.html
    • Loaded by the browser when the application starts
    • Contains the app-root tag from Angular
