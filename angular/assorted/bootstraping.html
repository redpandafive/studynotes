Bootstrapping

Non-Standalone

  • Loads main.ts file
    • This is referenced by the main property of angular.json to identify the startup
    • Enables Prod mode if that is enabled
    • Bootstraps AppModule with a call to platformBrowserDynamic().boostrapModule(AppModule)
  • AppModule bootstraps AppComponent
    • This is defined as a property on the NgModule decorator under the boostrap configuration property
    • Foundation component for an Angular project, so all applications require one and only one
  • App component
    • When you bootstrap with the AppComponent class (in main.ts), Angular looks for a app-root in the index.html, instantiates an instance of AppComponent, and renders it inside the app-root tag
    • The app-root tag will be in the Index.html file included in the root of the Src folder
  • Index.html
    • Loaded by the browser when the application starts
    • Contains the app-root tag from Angular

Standalone

  • Angular applications can be bootstrapped without any NgModule by using a standalone component as the application't root component
Use the bootstrapApplication API in the main.ts file
  • my-app.app.component is a standalone app module that hooks into the rest of the application
import {bootstrapApplication} from '@angular/platform-browser';
import {MyAppComponent} from './app/my-app.app.component';
bootstrapApplication(MyAppComponent);
You can pass a list of providers to the bootstrapApplication to configure providers for dependency injection
import {bootstrapApplication} from '@angular/platform-browser';
import {MyAppComponent} from './app/my-app.app.component';
bootstrapApplication(MyAppComponent, {
providers: [
{ provide: BACKEND_URL, useValue: 'https://myapp.mydomain.com/api' }
]
});

Angular includes a set of provide- prefixed functions to configure the app without needing to import NgModules.
  • Add router module and configure routes = provideRouter([arrayOfRouteObject])
  • Add HTTP Client module = provideHttpClient()
  • Many 3rd libraries include this configuration option - see Angular documentation if one does not
import {bootstrapApplication} from '@angular/platform-browser';
import {MyAppComponent} from './app/my-app.app.component';
bootstrapApplication(MyAppComponent, {
providers: [
{ provide: BACKEND_URL, useValue: 'https://myapp.mydomain.com/api' },
provideRouter([/* app routes */]),
]
});
