KeyValuePipe

  • Transforms an Object or Map into an array of key-value pairs
    • This allows you to iterate over an Object or Map within your template using a NgFor
  • Use when you need to:
    • Bind and iterate over a Map in a template
    • Bind to a NgFor loop to loop through an object's keys
  • Items are ordered by the key's datatype:
    • If strings, they are sorted in lexicographic order
    • If numbers, they are sorted in numeric order
  • Accepts one argument compareFn, which can be used to set the custom sort to the pipe
  • Result is an array of objects with two properties:
    • key
    • value

Syntax

<div *ngFor="let item of obj | keyvalue">
