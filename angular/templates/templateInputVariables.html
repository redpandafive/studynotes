Pass a Template as an Input Parameter

Start by defining the custom template for the buttons in the parent component


    <ng-template #customTabButtons>
      <div class="custom-class">
        <button class="tab-button" (click)="login()">
          {{loginText}}
        </button>
        <button class="tab-button" (click)="signUp()">
          {{signUpText}}
        </button>
      </div>
    </ng-template>
    <tab-container [headerTemplate]="customTabButtons"></tab-container>
On the TabContainer component (the child component),
  • Define an input property which is also a template named headerTemplate
  • If an input is provided, display that template.
  • Otherwise, display the defaultTabButtons template.
In the component...
export class TabContainerComponent {
@Input() headerTemplate: TemplateRef<any>
}
In the template
<ng-template #defaultTabButtons>
<div class="default-tab-buttons">
...
</div>
</ng-template>
<ng-container
*ngTemplateOutlet="headerTemplate ? headerTemplate: defaultTabButtons">
</ng-container>
... rest of tab container component ...

Can expand on this, create multiple inputs in the presentation component, and toggle between them based on some condition.
