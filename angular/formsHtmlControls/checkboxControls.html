Checkbox Controls

  • Provide a Null Option

Binding in a Template Form

The template, define the following
<input [(ngModel)]="isAccepted" type="checkbox" name="isAccepted" />
In the component, add the following
...
constructor() {}
isAccepted = true;
...
This example binds to a list of objects (each containing properties of id, name, and checked)
<ul>
<li *ngFor="let role of roles">
<input type="checkbox" [(ngModel)]="role.checked" />
<label>{{role.name}}</label>
