Form Validation Tips

Disable the submit button on a form until all FormControls in the FormGroup are valid
<button type="submit" [disabled]="myForm.invalid">Submit</button>
Add a check in teh submission function since it is easy to string out the attribute on the button
submitMyForm() { if (this.fg.invalid) { // here potentially add some visual feedback for a user return; } .... }
To perform a full recalculation and validation status of form controls, you can do something like the following in the submit function
Object.keys(this.activityFormGroup.controls).forEach(field => { let control = this.activityFormGroup.get(field); control.updateValueAndValidity(); });
