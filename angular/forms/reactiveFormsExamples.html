Reactive Forms - Examples

Basic Example

Setting up the Component and Template

Given a FormGroup property named profileForm
  • Must import ReactiveFormsModule from @angular/forms
    import { ReactiveFormsModule } from '@angular/forms';
  • In component, create a FormGroup property and assign the following to it. This will contain a FormControl instance for each control on the page.
    let firstName = new FormControl();
    let lastName = new FormControl();
    this.profileForm = new FormGroup({
    firstName: firstName,
    lastName: lastName
    });
  • Associate the component FormControls to the form control element in the template. This is done using the FormControlDirective
    • Add [formGroup]=”profileForm” to bind to the form element
    • For each element, specify the input each FormControl should correspond to. This adds a FormControlDirective to the element
    • The form control and DOM element now communicate with each other; updates to one updates the other
    • < id="name" type="text" [formControl]="name" >

Displaying Values in the Template

  • The following gives a snapshot of the current value. It changes as you update the form element.
    <p>Value: {{ name.value }}</p>

To Save Values

  • Add a ngSubmit event handler on the form element in the template
  • Define a saveProfile function on the component to handle the submit
  • If you would like for the inputs to have an initial value, you can provide that when you create the FormControls
    let lastName = new FormControl(currentUser.lastName) ...
