Reactive Forms - FormArray

FormArray

  • Allows you to manage any number of unnamed controls as an array
    • Opposed to FormGroup, which stores controls as properties
    • You can dynamically add or remove controls from form array instance
  • You can dynamically add or remove controls from form array instance
  • Can contain a FormGroup, FormControl, or another FormArray
  • This is a good option if you do not know the number of child values in advance
  • See Reactive Forms - Dynamic Forms
  • FormArray instance value and validation status is calculated from its child controls
  • Syntax
    • use the FormBuilder.array() method to create a new FormArray when using FormBuilder
    • use FormBuilder.control('') to push a new control to the array

Example

  • Create an instance using FormBuilder
    ...
    profileForm = this.fb.group({
    skillsForm: FormGroup;
    constructor(private fb:FormBuilder) {

    this.skillsForm = this.fb.group({
    name: '',
    skills: this.fb.array([]) ,
    });
    }
  • New FormGroup items can be added dynamically to the FormArray
    newSkill(): FormGroup {
    return this.fb.group({
    skill: '',
    exp: '',
    })
    }
    ...
    addSkills() {
    this.skills.push(this.newSkill());
    }
  • A getter method can be added to clean template code and type it
    get skills() : FormArray {
    return this.skillsForm.get("skills") as FormArray
    }
  • A getter method can be added to clean template code and type it
    get skills() : FormArray {
    return this.skillsForm.get("skills") as FormArray
    }
  • Within the form element bound to skillsForm, bind to the array using formArrayName and iterate using NgFor directive. Capture the current index of the iteration in the template local variable
    <div formArrayName="skills">
    <div *ngFor="let skill of skills().controls; let i=index">
    ...
    </div>
    </div>
  • Each element in the array is a FormGroup. It does not hae a name, so the index of the element is automatically assigned as the name. You can them assign the controls
    <div formArrayName="skills">
    <div *ngFor="let skill of skills().controls; let i=index">
    <div [formGroupName]="i">
    {{i}}
    skill name :
    <input type="text" formControlName="skill">
    exp:
    <input type="text" formControlName="exp">
    <button (click)="removeSkill(i)">Remove</button>
    </div> </div> </div>

Methods

  • controls
    • An array containing all teh controls that are part of the array
  • length
    • Total length of the array
  • at(index)
    • returns the form control at the given array position
  • push(FormControl)
    • Adds a new control to the end of the array
    • This could also be a FormGroup
  • removeAt(index)
    • Removes a control at a given index number of the array
  • getRawValue()
    • Gets the values of all form controls via the control.value property fo each control
