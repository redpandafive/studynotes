Reactive Forms - FormRecord

  • Tracks the value and validity state of a collection of FormControl instances, each of which has the same value type.
  • Useful when keys are not known ahead of time
  • Similar to FormGroup, except it can be used with a dynamic keys, with controls added and removed as needed.
    • Controls can be
      • Added
      • Replaced
      • Removed
  • Takes one generic argument, the type of the control it contains
