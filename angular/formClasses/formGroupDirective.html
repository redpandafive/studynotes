FormGroupDirective

  • Provides the following functionality:
    • Allows us to supply a FormGroup class to the template
    • Keeps track of when the form has been submitted
  • The selector for this directive is [formGroup]
  • It is exported as ngForm, so that is what you would use to get a reference to it in the template (see Example usage)
    • You can assign this to a template variable to get a reference to the FormGroupDirective in the template
    • This is useful if you want to want to display some content when the form is submitted
  • Example usage:
    • This example adds a template reference variable
    <form [formGroup]="myFormGroup" #form="ngForm">
