Reactive Forms - FormBuilder

  • Service that provides convenience methods for generating FormGroup and FormControls with less code

NonNullableFormBuilder

  • A new version of FormBuilder added with Typed Forms
  • It automatically adds {nonNullable: true} to each control without having to explicitly add it
  • Access it using the nonNullable proprety on the FormBuilder object
const fb = new FormBuilder();
const login = fb.nonNullable.group({
email: '',
password: '',
});

Setup

  • Import the service into your component
    import { FormBuilder } from '@angular/forms';
  • This is a service, so you must inject it into the component
    constructor(private fb: FormBuilder) { }
  • There are three methods on this service for generating form components
    • control()
      • Constructs a new FormControl
    • group()
      • Creates a new FormGroup instance
    • array()
      • Creates a new FormArray
  • Call the group() method to generate a FormGroup object
    • Pass an object to define the properties in the model
    • The value of each control name is an array containing the initial value as the first item in the array
      • Second and third values in the array will be synchronous and asynchronous validators
    • Example with a nested FormGroup:
      this.fb.group({
      firstName: [''],
      lastName: [''],
      address: this.fb.group({
      street: [''],
      city: [''],
      state: [''],
      zip: ['']
      }),
      });
    • Compare to the manual version
      profileForm = new FormGroup({
      firstName: new FormControl(''),
      lastName: new FormControl(''),
      address: new FormGroup({
      street: new FormControl(''),
      city: new FormControl(''),
      state: new FormControl(''),
      zip: new FormControl('')
      })
      });
