Styling Links

  • The RouterLinkActive directive can be added to add a space delimited string of CSS classes that will be added when the link is active
    • This applies when the link goes to the current URL
      code='<a [routerLink]="['/events']" routerLinkActive="active">My Link</a>
    • This applies the CSS class "active" when the link is active
    • This will match is the current route begins with the routerLink value
      • It will also match ['/events/new']
    • To get an exact match add routerLinkActiveOptions object
      code='<a [routerLink="'/events'" routerLinkActive="active" routerLinkActiveOptions="{exact:true}"]>All Events</a>
  • Anchor tags with no href attribute will not display a pointer when you hover over it
    • You can work around this by adding a CSS rule
      a { cursor: pointer}
