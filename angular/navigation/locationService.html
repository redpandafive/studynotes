Location Service

  • A service to interact with the browser URL
  • Inject Location service into a constructor to access it
  • Includes the following methods
    • back() - navigate back in history
    • forward() - navigate forward in history
  • It is better to use Router.navigate service to trigger route changes
  • Use Location only if you need to interact with or create normalized URLs outside of routing
    • Normalized against the application's base href
    • It is absolute from the URL host
    • Includes a leading slash, but no trailing slash
