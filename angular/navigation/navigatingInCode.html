Navigating in Code

  • There are two methods to do this available on the Router object
    • Navigate
      • References a route in the route table
        this.router.navigate(['courses', course.id])
    • navigateByUrl
      • references a URL
        this.router.navigateByUrl('/courses/${course.id}');
    • Must inject the Router object form @angular/router to do this
      • Import Router from @angular/router
      • Inject it into the constructor
    • Basic navigation to a defined route
      this.router.navigate(['/events'])
    • To pass query parameters with route, the second parameter takes an object
      • The resulting URL will be http://localhost:4200/books?orderby=price
        this.router.navigate(['/books'], { queryParams: { orderby: 'price' } })
    • QueryParamsHandling property
      • Preserve query parameters between URL requests
        this.router.navigate(['/books'], { queryParamsHandling: 'preserve' })
      • Calling this from http://localhost:4200/dvds?orderby=price will result in http://localhost:4200/books?orderby=price
    • Merge previous query parameters with any new ones
      • Calling this from http://localhost:4200/dvds?orderby=price will result in http://localhost:4200/books?orderby=price&type=kindle
        this.router.navigate(['/books'], {
        queryParams: { type: 'kindle' },
        queryParamsHandling: 'merge' })
