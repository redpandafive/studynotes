Computed Signals

  • A computed signal recomputes when its dependent signal(s) change.
  • These are readonly; cannot be updated with set, mutate, update
  • The computed value is memoized, meaning it stores the computed result
    • This computed value is reused next time the computed value is read, so it does not recalcuate on each read
This example is computed from two other signals and updates whenever one of them updates
exPrice = computed(() => this.selectedVehicle().price * this.quantity())
