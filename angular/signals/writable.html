Writable Signals

    Constructor

    Constructor

    • signal()
      • Use to create a writable signal (creates a WriteableSignal object)
      • You must supply a default starting value
    • signal()
      • Use to specify the type that should go into the signal
      • Type is optional when the default value can provide the type

    Reading a Signal

    • Signals have a getter function, so calling it reads thier value
      const count = signal(0);
      console.log('The count is: ' + count());

    Methods

    • Set()
      • Use to change the value of a writable signal
      • This replaces the value contained in the Signal. Consumers are then notified that the contents have changed.
      count.set(3);
    • Update()
      • Use to compute a new value based on the current value
        count.update(value => value + 1);
        quantity.update(qty => qty * 2);
    • Mutate()
      • When a signal contains an object or array, use this to mutate the object directly
      • This mutates the content in place, not the value itself
      • The following changes the first object in the array to 'done: true' without replacing it.
        const todos = signal([{title: 'Learn signals', done: false}]);
        todos.mutate(value => {
        value[0].done = true;
        });
