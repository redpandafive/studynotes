Linked Signals

  • Derived values that are re-evaluated when the state that it depends on changes, but can still be updated separately
      • You can set it later
      • If the dependent value changes, the linked signal will also reset with default functionality
This example is computed from two other signals and updates whenever one of them updates
exPrice = computed(() => this.selectedVehicle().price * this.quantity())
