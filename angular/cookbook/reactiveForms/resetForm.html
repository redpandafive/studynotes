Reset a Form

Reset method on a FormGroup will
  • Clear bound FormControl values
  • Reset the FormGroup
  • Mark all descendants pristine and untouched
  • Set the value of ALL descendants to their default values (or null if no defaults were provided)
this.fg.reset();
Pass an object with FormConrol keys mapped to values to reset to a specific form State
  • This could be values that were used to seed the form
  • Could also be a subset of the FormControls if desired
The following creates a common object of values that is used to seed the form and reset it back to those values
const profile = {firstname: 'first name'};
...
this.profileForm = this.formBuilder.group({
firstname: [this.profile.firstname],
});
...
this.profileForm.reset(this.profile);
