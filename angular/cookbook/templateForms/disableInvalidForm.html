Disable the Submit Button on an Invalid Tempplate Form

  • Tag the form controls with native HTML validtion attributes
    • Can also define custom validators if needed
    • https://developer.mozilla.org/en-US/docs/Web/Guide/HTML/Constraint_validation
    • https://angular.io/guide/form-validation#custom-validators
  • Create a local template variable to reference
    • The following captures a reference to a single control
      <input type="text" id="name" name="name" required minlength="4" [(ngModel)]="hero.name" #myVar="ngModel">
    • The following captures a reference to a form
      <form (ngSubmit)="onSubmit()" #myVar="ngForm"></form>
  • Bind to the disabled attribute on the button
    • <button [disabled]="myVar.invalid">Submit</button>
    • !myVar.valid will have the same result
