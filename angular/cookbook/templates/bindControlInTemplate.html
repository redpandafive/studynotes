Bind a Template Reference to a Control Within the Template

  • Add the ngModel directive to the control
    • Do not the [(ngModel)] syntax
    • Only want Angular to set a local model and update on change
    <input ngModelp>
  • Add a template reference to the element where you want to bind to
    • This gets a direct reference to the DOM element
    • The value from the input will be rebound each type you type
    <input ngModelp>
