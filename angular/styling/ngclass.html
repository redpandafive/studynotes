Class / NgClass

  • Use a class binding to add or remove a single class
  • Use ngClass when adding or removing multiple CSS classes simultaneously
  • A directive that adds and removes CSS class names to/from an elements class attribute
    • This can be done conditionally in templates
    • If desired effect can be done with browser CSS pseudo classes, use those instead

Class Bindings

  • Class bindings are better for single classes or single state styles
  • The syntax resembles a property binding
    • Start with prefix class
    • Follow with a dot
    • Follow with the name of a class
    • Enclose in brackets
  • Supply a Boolean expression
    • If it is evaluated to true, the class is added
    • If not, the class is not added
This will add the “green” class to the div if the event property has a property called time that contains a value of "8:00 am"
<div [class.green]="event.time === '8:00 am'"> ...

NgClass Directive

  • ngClass directive is better for multiple classes (multiple state styles)
  • The expression passed to a ngClass can be
    • An array
    • A string of class names
    • An object
    • Any expression that evaluates to a string, array, or object
      • This includes functions that return a value
  • You cannot use an if in a template since it is a statement (and templates do not allow statements)
  • You can use a ternary operator since it is an expression (it performs an assignment)

Passing an Array

  • All classes contained in the array will be assigned to the class attribute
The following will result in the following: ... class="btn btn-primary"...
<button [ngClass]="['btn', 'btn-primary']">Button</button>

Passing an String of Class Names

The following will result in the following: ... class="btn btn-primary"...
<button [ngClass]="'btn btn-primary']">Button</button>

Passing a Configuration Object

  • Each key of the object is a CSS class name to add
  • Each value should evaluate to either true (class is added) or false (class is removed)
Adds the “green” and “bold” classes if the time property contains a value of “8:00 am”
<div [ngClass]="{green: time === '8:00 am', Bold: time === '8:00 am'}"></div>"

Any Expression that Evaluates to a String, Array, or Object

This example uses a function on the component called getStartTimeClass
<div [ngClass]="getStartTimeClass()"></div>
The getStartTimeClass function can return an object with a property for each class and a Boolean value for each if they should be added
{green: true, Bold: true}
The getStartTimeClass function can instead return a space separated string containing all the classes that should be added
"green Bold"
The getStartTimeClass function can instead return an array of strings containing all the classes that should be added
["green", "Bold"]"
This example uses a ternary conditional since it is an expression
  • when val is greater than 10, apply class red. Else apply green.
<td [ngClass]="val > 10 ? 'red' : 'green'">{{ val }}</td>
