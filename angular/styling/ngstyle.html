Style Binding / NgStyle

  • You can also add a plain style attribute to the HTML element that will always be displayed
    • Any Angular bindings would be additive to this

Style Binding

  • The syntax is like property binding
    • Prefix with style
    • Follow with a dot
    • Follow with the name of a CSS style property
      • This can be either dash case or camel case
    • Can include a ternary template expression
The following adds a different color based on the canSave property
<button [style.background-color]="canSave ? 'cyan' : 'gray'">

NgStyle

  • Use NgStyle to set multiple inline styles simultaneously based on the state of the component
  • ngStyle can be bound to:
    • An object
      • Bind ngStyle to a key: value control object in the class
      • Each key of the control object is a style name
      • Each value should be the appropriate value for that style (including measurement if needed)
    • A function
      • The function should return an object with all rules
<div [ngStyle="getStartTimeStyle()"]>...</div>
In getStartTimeStyle method, return an object:
{ color: '#003300', 'font-weight': 'bold' }
