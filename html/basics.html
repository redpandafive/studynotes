HTML Basics

DOM Properties vs HTML Attributes

  • HTML attributes and DOM properties are not the same thing, event when they have the same name
  • HTML Attributes
    • defined by HTML
  • Properties
    • Defined by the DOM
  • There are some overlap in the naming between DOM properties and HTML attributes
    • There are some HTML attributes that don't have corresponding properties (such as colspan)
    • There are properties that don't have corresponding attributes (textContent is one)
    • There are attributes that appear to map to properties, but not in the way you expect
  • Attributes initialize DOM properties and then they are done
  • Property values can change; attribute values can't
    • HTML attribute specifies the initial value; DOM properties are the current value
  • Angular template binding works with properties and events, not attributes
