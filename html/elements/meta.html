Meta

  • Metadata about the HTML document
  • Includes character encoding, description
  • Information about your website can be embedded here so sites like FaceBook, Twitter, and YouTube can read it
    • This is used to generate preview cards and other embedded previews
    • Look if Open Graph values used in the property attribute of the meta tag for more information
