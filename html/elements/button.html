Button

Useful attributes
  • formaction attribute
    • Specifies where to send form data when a form is submitted
    • This will override the action attribute on a form tag
    • Should only be applied to input or button tags
    • Can only be used for type submit and image
    • Use classes
      • Useful if you have multiple buttons on a form, each with different destinations
      • If you have a table of data and you want to have a delete button on each row. This allows you to post from the button by assigning a different id to each row.
