Picture

  • Allows you to have multiple versions of an image and display one of them based on a condition, such as screen width
    • This is good to use when you have a full size image to display for desktops and a smaller logo to display for mobile screens
  • Image tag should be the very last element within the picture element
    • It is only loaded and displayed if the source elements before it do not have a matching condition
  • Source tags can be used to specify conditional images
    • Attributes includes
      • type attribute = matches if the browser supports an image type (such as image/webp)
      • media attribute = matches if the supplied media query matches
    • Multiple source tags can be supplied. The first match is the only one selected.
    • If none match, the img tag is selected by default
The following will display the Logo-Grey.png when the screen width is at least 900px. Otherwise, it will show the logo.png image.
<picture>
<source srcset="Logo-Grey.png" media="(min-width: 900px)" >
<img src="logo.png" style="width: 700px" > </picture>
