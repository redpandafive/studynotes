Lists

  • Ordered lists = <ol>
    • Display numbers next to list items
    • Contain list items
  • Unordered lists = = <ul>
    • Display bullet points next to list items
  • Menu lists = = <menu>
    • Semantic alternative to ul
      • Treated by browsers no different than ul
      • Ul is primiarly for items for display, while menu was indended for interactive items
    • Represent an unordered list of items
    • An example would be a list of social media share buttons
    • Display bullet points next to list items
  • Description lists = = <dl>
    • Used for a glossary or to display metadata (key-value pairs)
    • Do not use list markers
    • Use Description Terms (<dt>) and Description Details (<dd>)
    • https://developer.mozilla.org/en-US/docs/Web/HTML/Element/dl

List Markers

  • Symbol to the left of a list item
  • Typically a number or bullet
  • Use the ::marker pseudo element to style

Cookbook

Remove list markers from ol, ul, or menu
ol, ul, menu {
list-style: none;
}
