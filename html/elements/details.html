Detail / Summary

  • Creates a widget in which information is only visible when the widget it toggled to an open state
  • If in a closed state, the contents of the enclosed summary element are only shown
  • Typically includes a triangle icon on the left side of the summary to indicate state of the widget
  • Attributes
    • open
      • Boolean attribute
      • Indicates if the contents of the details element are current visible
      • The attribute must completely removed to be false - cannot just set it to false
<details>
<summary>Details</summary>
Something small enough to escape casual notice.
</details>
