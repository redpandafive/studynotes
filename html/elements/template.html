Template

  • Allows you to define a template within the HTML of the page
  • This can contain any content, including HTML elements
  • By default, this is not displayed in the page
  • You can use JS to select it and append it to the page
  • This is useful when you have content that needs to be repeated with small modifications that you can do through JS
<template>
<h5>
This is a template
</h5>
<small>
Hi
</small>
</template>
....
<script>
const template = document.querySelector('template');
const content = template.content.cloneNode(true);
content.querySelector('small').innerText = 'Modified test';
document.body.append(content);
</script>
