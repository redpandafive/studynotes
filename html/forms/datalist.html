Data List Element

  • Contains a set of option elements that represent the permissible or recommended options availabe to choose from within other controls
  • To bind it to a control
    • Give the data list a unique identifier in the id attribute
    • Add the list attribute to the input element with the same identifier as value

Examples


<label for="ice-cream-choice">Choose a flavor:</label>
<input list="ice-cream-flavors" id="ice-cream-choice" name="ice-cream-choice">
<datalist id="ice-cream-flavors">
<option value="Chocolate">
<option value="Coconut">
<option value="Mint">
<option value="Strawberry">
<option value="Vanilla">
</datalist>
