Web Components

  • Consist of three Components
    • Custom Elements
      • Allow you to define your web component, then use it with a custom HTML tag to include it in pages
    • HTML Templates
      • Not actually displayed in the page; they are cloned and inserted into the page
      • Contains markup that will be bundled into the component
    • Shadow DOM
      • The encapsulated DOM that you have access to within a web component
      • This keeps the web component logic from leaking out into the light DOM (main DOM)
