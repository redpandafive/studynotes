Filling Out Forms

  • HTML Text Inputs
    • Use a selector to get the IWebElement for the control
      code='IWebElement textField = driver.FindElement(By.Id("Firstname"));'
    • You cannot write to the Text property of the IWebElement - it is read only
    • Use the SendKeys to simulate a user typing into the text box
      code='textField.SendKeys("Colin");'
    • This can be simplified to
      code='driver.FindElement(By.Id("Firstname")).SendKeys("Colin");'
  • HTML Radio Buttons
    • Use Developer Tools to locate the target radio button
    • Use the FindElement method to locate the IWebElement
    • Call the click method on the IWebElement
      code='driver.FindElement(By.Id("Single")).Click();'
  • HTML Select Options
    • Locate the select options list using Developer Tools
    • Get an IWebElement reference to the Select control
      code='IWebElement selectElement = driver.FindElement(By.Id("BusinessDropDown"));'
    • Define an instance of a SelectElement object passing the IWebElement instance as a constructor parameter
      SelectElement mySelect = new SelectElement(selectElement);"
    • The SelectElement has several methods and properties you can reference in your tests, including
      • Options - list of IWebElements representing all the options in the select. You can iterate over this to work with each option object.
      • SelectedOption - IWebElement for the selected option. You can then reference the Text property to see option text
      • SelectByIndex(idx) - uses a zero based index to specify an Option object
      • SelectByText("text_to_get") - selects the Option with the specified text (value displayed on the screen)
      • SelectByValue("value_to_get") - selects the Option with the specified value
  • HTML Checkbox
    • code='driver.FindElement(By.Id("my_checkbox")).Click();'
  • Submitting a Form
    • First way is to call click on the submit button for the form
      code='driver.FindElement(By.Id("submit_button")).Click();'
    • Another way is to select one of the controls from the form and call the Submit Method on the IWebElement
      code='driver.FindElement(By.Id("my_checkbox")).Submit();'

Verifying Form Submission

  • Things to check in assert statements for test
    • Verify title is the correct page (submit confirmation page)
    • Grab elements from the confirmation screen and compare them to the inputted values
    • Verify auto-generated values that are system generated are present - can't really check accuracy of these

Testing Form Validation

  • You can define tests that verify validation is working as expected
  • Create tests that deliberately add invalid values into the form and try submitting it
  • Grab the validation block HTML and check the number of errors and the text of the errors to see if they match the expected result
  • Fix the invalid values to valid values and try submitting again
